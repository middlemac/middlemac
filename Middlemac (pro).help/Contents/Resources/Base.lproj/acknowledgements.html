Acknowledgements and License

Acknowledgements

Middlemac depends entirely on Middleman, without which this work would not be practical.

We have chosen the name Middlemac in homage to Middleman and this choice should not be inferred to mean that this project is related to Middleman.

We have designed a logo very similar to that of the Middleman project. It should not be inferred that Middlemac is related to the Middleman project based on this design choice. We will happily change to logo if anyone from the official Middleman project makes such a request.

Middleman itself depends on a whole ecosystem of free and open source software. Please visit http://middlemanapp.com in order to have a full understanding of all of the pieces that make Middlemac work. Without all of these contributions, Middleman and Middlemac would not be possible.
