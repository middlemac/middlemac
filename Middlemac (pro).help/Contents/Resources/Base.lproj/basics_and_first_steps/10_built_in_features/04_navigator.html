Navigator

Learn how the page navigator can make navigation easy for your users, as well as how to turn it on or off for groups and individual pages.

Near the bottom of this page you can see the navigator. It’s visible because this page’s parent index.html file includes navigate: true in its frontmatter.

Setting :navigate in the frontmatter will cause all pages with an order to display the navigator. This can be overridden on a page-per-page basis by using the :navigator frontmatter key, e.g., navigator: false.

The partial prev_next provides the navigator structure, and the appearance is set in CSS.
