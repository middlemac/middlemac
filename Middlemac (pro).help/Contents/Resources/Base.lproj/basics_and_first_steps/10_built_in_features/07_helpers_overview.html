Helpers overview

A brief look at how Middlemac’s Helpers can help you out.

Middleman has several useful, built-in helpers documented on its web site and in this document’s reference section.

Middlemac introduces several other helpers that are available in your files.

cfBundleIdentifier
Returns the product CFBundleIdentifier for the current target.
cfBundleName
Returns the product CFBundleName for the current target.
product_name
Returns the ProductName for the current target.
product_uri
Returns the ProductURI for the current target.
target_name
Return the current build target.
target_name?(proposal)
Is the current target proposal?
target_feature?(proposal)
Does the target have the feature feature?
