Configure your help project

You will configure your helpbook project in config.rb, which will specify the settings that Middlemac uses to build one or more valid Apple Helpbooks.

The file config.rb has settings which both affect how Middleman operates as well as how Middlemac builds your helpbooks. If you follow the Middlemac conventions, there should be no reason to change the Middleman settings.

When you open config.rb in your text editor, near the top you will readily identify the start of the Middlemac configuration block, which is delimited by this block at the beginning:

################################################################
# Configuration. Change the option values to suit your needs.
################################################################

And terminated by this block at the end:

################################################################
# STOP! There's nothing below here that you should have to
# change. Just follow the conventions and framework provided.
################################################################

This file is very well documented, and as such not much is to be gained by repeating its content here, although much more information is available in the reference section.
