Middlemac 1.0

Middlemac is the specialized Apple Help building system that uses the tools of Middleman to make building Apple Helpbooks for your Mac OS X applications a snap. Whether you are targeting multiple versions of your application or a single version, once properly (simply!) configured, Middlemac will take all of the pain out of building Help files.
