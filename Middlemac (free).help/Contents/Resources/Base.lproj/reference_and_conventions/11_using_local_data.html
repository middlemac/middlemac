Using local data

You can use a local file as a datasource to generate data-based pages automatically. Learn how in this reference section.

You can use data from external sources with Middleman and Middlemac, too. In assets/data/ a sample globals.yml contains some sample data in YAML format. Letâ€™s see what it contains:

  • A sample work of Douglas Adams is The Restaurant at the End of the Universe.
  • A sample work of Edward Bulwer-Lytton is Paul Clifford.
  • A sample work of Daniel Defoe is Robinson Crusoe.

These previous sentences were generated with this ERB code:

<% data.globals.famous_writers.each do |writer| %>
- A sample work of <%= writer.author %> is _<%= writer.sample_work %>_.
<% end %>

Accessing data

This data is available to you via data.file_name.top_level_key.
