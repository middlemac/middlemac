Middlemac command line overview

Learn the basics of using the middlemac.rb command-line tool.

Middlemac command signature

./middlemac.rb TARGET... | all [--verbose | -v] [--quiet | -q] [--help | -h] [--server | -s]

Getting help and identifying valid targets

If you run Middlemac with an invalid target, with no parameters, or with the -h or --help switch, Middlemac will display its documentation.

General

You can run Middlemac in server mode by specifying a target and the -s switch.

You can build one or more targets by specifying one or more target names, or all to build all targets.
